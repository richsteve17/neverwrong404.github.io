# App Icons

Place your app icons in this directory with the following sizes:

- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-180.png (180x180) - iOS home screen
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

You can generate these icons from a single 512x512 image using online tools like:
- https://realfavicongenerator.net/
- https://www.pwa-icon-generator.com/
