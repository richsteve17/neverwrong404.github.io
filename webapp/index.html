Setup Instructions

Quick Setup (5 minutes)

1. Get Google OAuth Client ID

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable the Gmail API
  4. Create OAuth 2.0 Client ID (Web application)
  5. Add authorized JavaScript origin: https://yourusername.github.io
  6. Copy your Client ID

2. Get Gemini API Key

  1. Visit Google AI Studio
  2. Create a new API key
  3. Copy the API key

3. Configure the App

  1. Open js/config.js in the code
  2. Replace YOUR_GOOGLE_CLIENT_ID with your Client ID
  3. Replace YOUR_GEMINI_API_KEY with your API key
  4. Save and reload the page

4. Add to Home Screen (iOS)

  1. Tap the Share button in Safari
  2. Scroll down and tap "Add to Home Screen"
  3. Tap "Add"
  4. Open from your home screen!
